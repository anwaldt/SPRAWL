## OSC Messages

### Azimuth

    /source/azim i f

- first argument: source index to control (starting with 0)
- second argument: azimuth value (-pi/2 ... pi/2)


### Elevation

    /source/elev i f

- first argument: source index to control (starting with 0)
- second argument: elevation value (-pi/2 ... pi/2)
